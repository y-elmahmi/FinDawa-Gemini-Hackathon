## ğŸ“– Overview
**FinDawa** is a revolutionary healthcare platform designed to solve the critical issue of medication shortages in Morocco. 

By leveraging **Google Gemini 3's multimodal capabilities**, FinDawa does not just locate pharmacies; it analyzes sales data to **predict stockouts before they happen**, alerting pharmacists to restock essential life-saving drugs.

## âš™ï¸ Getting Started

To run FinDawa locally on your machine:

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/y-elmahmi/FinDawa-Gemini-Hackathon.git](https://github.com/y-elmahmi/FinDawa-Gemini-Hackathon.git)
    cd FinDawa-Gemini-Hackathon
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Set up Environment Variables**
    Create a `.env` file in the root directory and add your API Keys:
    ```env
    VITE_GEMINI_API_KEY=your_google_gemini_key
    VITE_SUPABASE_URL=your_supabase_url
    VITE_SUPABASE_ANON_KEY=your_supabase_key
    ```

4.  **Run the App**
    ```bash
    npm run dev
    ```
